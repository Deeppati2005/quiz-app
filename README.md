## Introduction
The Quiz App is a simple and interactive application that allows users to take quizzes on various topics. This project is developed using Java and is designed to provide a seamless user experience with a straightforward interface.

## Features
- **User Authentication:** Secure user login and registration.
- **Quiz Management:** Create, update, delete, and view quizzes.
- **Question Management:** Add, update, delete, and view questions within quizzes.
- **Timer:** Each quiz comes with a timer to enhance the challenge.
- **Score Calculation:** Automatic calculation of scores based on user responses.
- **Responsive Design:** Works well on different screen sizes and devices.

## Technologies Used
- **Java:** The core programming language used for the application.
- **Spring Boot:** For building the backend services.
- **Thymeleaf:** For rendering the front-end views.
- **H2 Database:** For storing user and quiz data.
- **Maven:** For project management and dependency management.

## Installation
Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Deeppati2005/quiz-app.git
   cd quiz-app
   ```

2. **Install dependencies:**
   Make sure you have Maven installed. Then run:
   ```bash
   mvn clean install
   ```

3. **Run the application:**
   ```bash
   mvn spring-boot:run
   ```

4. **Access the application:**
   Open your browser and go to `http://localhost:8080`.

## Usage
### User Authentication
- **Register:** Go to the registration page and create a new account.
- **Login:** Use your credentials to log in to the application.

### Managing Quizzes
- **Create Quiz:** Navigate to the quiz management section and create a new quiz.
- **Edit Quiz:** Select a quiz to edit its details or questions.
- **Delete Quiz:** Remove a quiz that is no longer needed.
- **Take Quiz:** Start a quiz and answer the questions within the given time.

### Viewing Results
- After completing a quiz, the score is calculated automatically, and you can view your results immediately.
